Push your latest commits to GitHub so your code is backed up and shareable. This does NOT deploy to production.

Steps:

1. Run git status to check for uncommitted changes. If there are any, run /snapshot first to commit them.
2. Push to GitHub: `git push origin main`
3. Confirm success and show the GitHub repository URL.
4. Remind the user: this only backs up your code to GitHub. To deploy to production, run /deploy.
